# Dashboard de Vendas | Power BI

Projeto de Business Intelligence desenvolvido com dados 100% fictícios, criado para simular um ambiente corporativo real de análise de vendas e tomada de decisão.

O projeto aplica modelagem dimensional (Star Schema), métricas de negócio em DAX e um dashboard executivo no Power BI, voltado à tomada de decisão.

## Objetivos
- Analisar desempenho de vendas ao longo do tempo
- Avaliar produtos, vendedores, canais e regiões
- Medir impacto de descontos na receita e margem

## Tecnologias
- Power BI
- DAX
- Modelagem Dimensional
- GitHub

## Status do Projeto
 Em desenvolvimento – fase de modelagem e construção do dashboard.

## Observação
Todos os dados utilizados são fictícios e foram criados exclusivamente para fins educacionais e de portfólio.
